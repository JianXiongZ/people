This is a script for upgrade Avalon6 controller and machine.
Note: please take care before your usage.

cmd.txt: The commands run on the Avalon6 controller.
ip.txt: The controller's ip address which are to be upgrade.
local_cmd.bat: The commands run on windows.
plink.exe: The program use to connect the Avalon6 controller.
start_upgrade.bat: The key file. Click it to upgrade all controllers and machines.
